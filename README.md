Установить все нужные библиотеки из файла requirements.txt.
После запуска авторизовать аккаунт с которого будут происходить перепродажи, далее выбрать режим и настроить минимальную, максимальную цену, процент.
В extensions.txt вводим названия подарков исключений, т.е которые не будут покупаться и проверяться. В offers.csv вводим название подарка, модель, фон (если нет, то None), максимальную цену предложения и глубину. по этому списку будут отправляться личные оффера.
все остальные инструкции будут в самом скрипте.
